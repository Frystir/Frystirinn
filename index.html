Hvernig á að tengja Firebase:
1. Farðu á console.firebase.google.com
2. Búðu til nýtt verkefni (Add project)
3. Veldu Firestore Database → Start in test mode
4. Farðu í Project Settings → Your apps → Web app (</>)
5. Afritaðu firebaseConfig gildin
6. Opnaðu þennan HTML skrá í textaritli
7. Fylltu út FIREBASE_CONFIG efst í skriptunni
8. Settu sama skrána upp á báða síma (t.d. í Google Drive)
Install as app
iOS Safari Tap Share → "Add to Home Screen" → Add
Android Chrome Tap ⋮ → "Add to Home screen" → Add
